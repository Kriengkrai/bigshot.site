 17 /**
 18  * Creates a new VR panorama in a canvas. <b>Requires WebGL support.</b>
 19  * (Note: See {@link bigshot.VRPanorama#dispose} for important information.)
 20  * 
 21  * <h3 id="creating-a-cubemap">Creating a Cube Map</h3>
 22  *
 23  * <p>The panorama consists of six image pyramids, one for each face of the VR cube.
 24  * Due to restrictions in WebGL, each texture tile must have a power-of-two (POT) size -
 25  * that is, 2, 4, ..., 128, 256, etc. Furthermore, due to the way the faces are tesselated
 26  * the largest image must consist of POT x POT tiles. The final restriction is that the 
 27  * tiles must overlap for good seamless results.
 28  *
 29  * <p>The MakeImagePyramid has some sensible defaults built-in. If you just use the
 30  * command line:
 31  *
 32  * <code><pre>
 33  * java -jar bigshot.jar input.jpg temp/dzi \
 34  *     --preset dzi-cubemap \ 
 35  *     --format folders
 36  * </pre></code>
 37  * 
 38  * <p>You will get 2034 pixels per face, and a tile size of 256 pixels with 2 pixels
 39  * overlap. If you don't like that, you can use the <code>overlap</code>, <code>face-size</code>
 40  * and <code>tile-size</code> parameters. Let's take these one by one:
 41  *
 42  * <ul>
 43  * <li><p><code>overlap</code>: Overlap defines how much tiles should overlap, just to avoid
 44  * seams in the rendered results caused by finite numeric precision. The default is <b>2</b>, which
 45  * I've found works great for me.</p></li>
 46  * <li><p><code>tile-size</code>: First you need to decide what POT size the output should be.
 47  * Then subtract the overlap value. For example, if you set overlap to 1, <code>tile-size</code>
 48  * could be 127, 255, 511, or any 2<sup>n</sup>-1 value.</p></li>
 49  * <li><p><code>face-size</code>: Finally, we decide on a size for the full cube face. This should be
 50  * tile-size * 2<sup>n</sup>. Let's say we set n=3, which makes each face 8x8 tiles at the most zoomed-in
 51  * level. For a tile-size of 255, then, face-size is 255*2<sup>3</sup> = 255*8 = <b>2040</b>.</p></li>
 52  * </ul>
 53  * 
 54  * <p>A command line for the hypothetical scenario above would be:
 55  * 
 56  * <code><pre>
 57  * java -jar bigshot.jar input.jpg temp/dzi \
 58  *     --preset dzi-cubemap \ 
 59  *     --overlap 1 \
 60  *     --tile-size 255 \
 61  *     --face-size 2040 \
 62  *     --format folders
 63  * </pre></code>
 64  *
 65  * <p>If your tile size numbers don't add up, you'll get a warning like:
 66  *
 67  * <code><pre>
 68  * WARNING: Resulting image tile size (tile-size + overlap) is not a power of two: 255
 69  * </pre></code>
 70  *
 71  * <p>If your face size don't add up, you'll get another warning:
 72  *
 73  * <code><pre>
 74  * WARNING: face-size is not an even multiple of tile-size: 2040 % 254 != 0
 75  * </pre></code>
 76  *
 77  * <h3 id="integration-with-saladoplayer">Integration With SaladoPlayer</h3>
 78  *
 79  * <p><a href="http://panozona.com/wiki/">SaladoPlayer</a> is a cool
 80  * Flash-based VR panorama viewer that can display Deep Zoom Images.
 81  * It can be used as a fallback for Bigshot for browsers that don't
 82  * support WebGL.
 83  *
 84  * <p>Since Bigshot can use a Deep Zoom Image (DZI) via a {@link bigshot.DeepZoomImageFileSystem}
 85  * adapter, the common file format is DZI. There are two cases: The first is
 86  * when the DZI is served up as a folder structure, the second when
 87  * we pack the DZI into a Bigshot archive and serve it using bigshot.php.
 88  *
 89  * <h4>Serving DZI as Folders</h4>
 90  *
 91  * <p>This is an easy one. First, we generate the required DZIs:
 92  *
 93  * <code><pre>
 94  * java -jar bigshot.jar input.jpg temp/dzi \
 95  *     --preset dzi-cubemap \ 
 96  *     --format folders
 97  * </pre></code>
 98  * 
 99  * <p>We'll assume that we have the six DZI folders in "temp/dzi", and that
100  * they have "face_" as a common prefix (which is what Bigshot's MakeImagePyramid
101  * outputs). So we have, for example, "temp/dzi/face_f.xml" and the tiles for face_f
102  * in "temp/dzi/face_f/". Set up Bigshot like this:
103  *
104  * <code><pre>
105  * bvr = new bigshot.VRPanorama (
106  *     new bigshot.VRPanoramaParameters ({
107  *             container : document.getElementById ("canvas"),
108  *             basePath : "temp/dzi",
109  *             fileSystemType : "dzi"
110  *         }));
111  * </pre></code>
112  * 
113  * <p>SaladoPlayer uses an XML config file, which in this case will
114  * look something like this:
115  * 
116  * <code><pre>
117  * <SaladoPlayer>
118  *     <global debug="false" firstPanorama="pano"/>
119  *     <panoramas>
120  *         <panorama id="pano" path="temp/dzi/face_f.xml"/>
121  *     </panoramas>
122  * </SaladoPlayer>
123  * </pre></code>
124  *
125  * <h4>Serving DZI as Archive</h4>
126  *
127  * <p>This one is a bit more difficult. First we create a DZI as a bigshot archive:
128  *
129  * <code><pre>
130  * java -jar bigshot.jar input.jpg temp/dzi.bigshot \
131  *     --preset dzi-cubemap \ 
132  *     --format archive
133  * </pre></code>
134  *
135  * <p>We'll assume that we have our Bigshot archive at
136  * "temp/dzi.bigshot". For this we will use the "entry" parameter of bigshot.php
137  * to serve up the right files:
138  *
139  * <code><pre>
140  * bvr = new bigshot.VRPanorama (
141  *     new bigshot.VRPanoramaParameters ({
142  *             container : document.getElementById ("canvas"),
143  *             basePath : "/bigshot.php?file=temp/dzi.bigshot&entry=",
144  *             fileSystemType : "dzi"
145  *         }));
146  * </pre></code>
147  * 
148  * <p>SaladoPlayer uses an XML config file, which in this case will
149  * look something like this:
150  * 
151  * <code><pre>
152  * <SaladoPlayer>
153  *     <global debug="false" firstPanorama="pano"/>
154  *     <panoramas>
155  *         <panorama id="pano" path="/bigshot.php?file=dzi.bigshot&amp;entry=face_f.xml"/>
156  *     </panoramas>
157  * </SaladoPlayer>
158  * </pre></code>
159  *
160  * <h3>Usage example:</h3>
161  * @example
162  * var bvr = new bigshot.VRPanorama (
163  *     new bigshot.VRPanoramaParameters ({
164  *             basePath : "/bigshot.php?file=myvr.bigshot",
165  *             fileSystemType : "archive",
166  *             container : document.getElementById ("bigshot_canvas")
167  *         }));
168  * @class A cube-map VR panorama.
169  *
170  * @param {bigshot.VRPanoramaParameters} parameters the panorama parameters.
171  *
172  * @see bigshot.VRPanoramaParameters
173  */
174 bigshot.VRPanorama = function (parameters) {
175     var that = this;
176     
177     this.parameters = parameters;
178     this.maxTextureMagnification = parameters.maxTextureMagnification;
179     this.container = parameters.container;
180     this.browser = new bigshot.Browser ();
181     this.dragStart = null;
182     this.dragDistance = 0;
183     this.hotspots = [];
184     
185     this.transformOffsets = {
186         yaw : parameters.yawOffset,
187         pitch : parameters.pitchOffset,
188         roll : parameters.rollOffset
189     };
190     
191     /**
192      * Current camera state.
193      * @private
194      */
195     this.state = {
196         /**
197          * Pitch in degrees.
198          * @private
199          */
200         p : 0.0,
201         
202         /**
203          * Yaw in degrees.
204          * @private
205          */
206         y : 0.0,
207         
208         /**
209          * Field of view (vertical) in degrees.
210          * @private
211          */
212         fov : 45,
213         
214         tx : 0.0,
215         
216         ty : 0.0,
217         
218         tz : 0.0
219     };
220     
221     /**
222      * Renderer wrapper.
223      * @private
224      * @type bigshot.VRRenderer
225      */
226     this.renderer = null;
227     if (this.parameters.renderer) {
228         if (this.parameters.renderer == "css") {
229             this.renderer = new bigshot.CSS3DVRRenderer (this.container);
230         } else if (this.parameters.renderer == "webgl") {
231             this.renderer = new bigshot.WebGLVRRenderer (this.container)
232         } else {
233             throw new Error ("Unknown renderer: " + this.parameters.renderer);
234         }
235     } else {
236         this.renderer = 
237             bigshot.webglutil.isWebGLSupported () ? 
238         new bigshot.WebGLVRRenderer (this.container)
239         :
240         new bigshot.CSS3DVRRenderer (this.container);
241     }

243     /**
244      * Adds a hotstpot.
245      *
246      * @param {bigshot.VRHotspot} hs the hotspot to add
247      */
248     this.addHotspot = function (hs) {
249         this.hotspots.push (hs);
250     }

252     /**
253      * Returns the {@link bigshot.VRPanoramaParameters} object used by this instance.
254      *
255      * @type bigshot.VRPanoramaParameters
256      */
257     this.getParameters = function () {
258         return this.parameters;
259     }
260     
261     this.setTranslation = function (x, y, z) {
262         this.state.tx = x;
263         this.state.ty = y;
264         this.state.tz = z;
265     }
266     
267     this.getTranslation = function () {
268         return {
269             x : this.state.tx,
270             y : this.state.ty,
271             z : this.state.tz
272         };
273     }

275     /**
276      * Sets the field of view.
277      *
278      * @param {number} fov the vertical field of view, in degrees
279      */
280     this.setFov = function (fov) {
281         fov = Math.min (this.parameters.maxFov, fov);
282         fov = Math.max (this.parameters.minFov, fov);
283         this.state.fov = fov;
284     }
285     
286     /**
287      * Gets the field of view.
288      *
289      * @return {number} the vertical field of view, in degrees
290      */
291     this.getFov = function () {
292         return this.state.fov;
293     }
294     
295     this.snapPitch = function (p) {
296         p = Math.min (this.parameters.maxPitch, p);
297         p = Math.max (this.parameters.minPitch, p);
298         return p;
299     }
300     
301     /**
302      * Sets the current camera pitch.
303      *
304      * @param {number} p the pitch, in degrees
305      */
306     this.setPitch = function (p) {
307         this.state.p = this.snapPitch (p);
308     }
309     
310     /**
311      * Subtraction mod 360, sort of...
312      * @returns the angular distance with smallest magnitude to add to p0 to get to p1 % 360
313      */
314     this.circleDistance = function (p0, p1) {
315         if (p1 > p0) {
316             // p1 is somewhere clockwise to p0
317             var d1 = (p1 - p0); // move clockwise
318             var d2 = ((p1 - 360) - p0); // move counterclockwise, first -p0 to get to 0, then p1 - 360.
319             return Math.abs (d1) < Math.abs (d2) ? d1 : d2;
320         } else {
321             // p1 is somewhere counterclockwise to p0
322             var d1 = (p1 - p0); // move counterclockwise
323             var d2 = (360 - p0) + p1; // move clockwise, first (360-p= to get to 0, then another p1 degrees
324             return Math.abs (d1) < Math.abs (d2) ? d1 : d2;
325         }
326     };
327     
328     /**
329      * Subtraction mod 360, sort of...
330      * @private
331      */
332     var circleSnapTo = function (p, p1, p2) {
333         var d1 = this.circleDistance (p, p1);
334         var d2 = this.circleDistance (p, p2);
335         return Math.abs (d1) < Math.abs (d2) ? p1 : p2;
336     };
337     
338     this.snapYaw = function (y) {
339         y %= 360;
340         if (y < 0) {
341             y += 360;
342         }
343         if (this.parameters.minYaw < this.parameters.maxYaw) {
344             if (y > this.parameters.maxYaw || y < this.parameters.minYaw) {
345                 y = circleSnapTo (y, this.parameters.minYaw, this.parameters.maxYaw);
346             }
347         } else {
348             // The only time when minYaw > maxYaw is when the interval
349             // contains the 0 angle.
350             if (y > this.parameters.minYaw) {
351                 // ok, we're somewhere between minYaw and 0.0
352             } else if (y > this.parameters.maxYaw) {
353                 // we're somewhere between maxYaw and minYaw 
354                 // (but on the wrong side).
355                 // figure out the nearest point and snap to it
356                 y = circleSnapTo (y, this.parameters.minYaw, this.parameters.maxYaw);
357             } else {
358                 // ok, we're somewhere between 0.0 and maxYaw
359             }
360         }
361         return y;
362     }

364     /**
365      * Sets the current camera yaw. The yaw is normalized between
366      * 0 <= y < 360.
367      *
368      * @param {number} y the yaw, in degrees
369      */
370     this.setYaw = function (y) {
371         this.state.y = this.snapYaw (y);
372     }

375      * Gets the current camera yaw.

384      * Gets the current camera pitch.
385      *
386      * @return {number} the pitch, in degrees

388     this.getPitch = function () {
389         return this.state.p;

392     /**
393      * Unregisters event handlers and other page-level hooks. The client need not call this
394      * method unless bigshot images are created and removed from the page
395      * dynamically. In that case, this method must be called when the client wishes to
396      * free the resources allocated by the image. Otherwise the browser will garbage-collect
397      * all resources automatically.
398      * @public
399      */
400     this.dispose = function () {
401         this.browser.unregisterListener (window, "resize", this.onresizeHandler, false);
402     };

404     /**
405      * Sets up transformation matrices etc.
406      */
407     this.beginRender = function (cause, data) {
408         this.onrender (this.ONRENDER_BEGIN, cause, data);
409         this.renderer.beginRender (this.state.y, this.state.p, this.state.fov, this.state.tx, this.state.ty, this.state.tz, this.transformOffsets.yaw, this.transformOffsets.pitch, this.transformOffsets.roll);
410     }
411     
412     this.renderListeners = new Array ();
413     
414     /**
415      * Add a function that will be called at the end of every render.
416      *
417      * @param {function(state)} listener
418      */
419     this.registerRenderListener = function (listener) {
420         var rl = new Array ();
421         rl.concat (this.renderListeners);
422         rl.push (listener);
423         this.renderListeners = rl;
424     };
425     
426     /**
427      * Removes a function that will be called at the end of every render.
428      *
429      * @param {function()} listener
430      */
431     this.unregisterRenderListener = function (listener) {
432         var rl = new Array ();
433         rl.concat (this.renderListeners);
434         for (var i = 0; i < rl.length; ++i) {
435             if (rl[i] == listener) {
436                 rl = rl.splice (i, 1);
437                 break;
438             }
439         }
440         this.renderListeners = rl;
441     };
442     
443     this.ONRENDER_BEGIN = 0;
444     this.ONRENDER_END = 1;
445     this.ONRENDER_TEXTURE_UPDATE = 0;
446     
447     /**
448      * Called at the start and end of every render.
449      *
450      * @event
451      * @private
452      * @type function()
453      */
454     this.onrender = function (state, cause, data) {
455         var rl = this.renderListeners;
456         for (var i = 0; i < rl.length; ++i) {
457             rl[i](state, cause, data);
458         }
459     };
460     
461     /**
462      * Performs per-render cleanup.
463      */
464     this.endRender = function (cause, data) {
465         for (var f in this.vrFaces) {
466             this.vrFaces[f].endRender ();
467         }
468         this.renderer.endRender ();
469         this.onrender (this.ONRENDER_END, cause, data);
470     }
471     
472     /**
473      * Renders the VR cube.
474      */
475     this.render = function (cause, data) {
476         this.beginRender (cause, data);
477         
478         var scene = this.renderer.createTexturedQuadScene ();
479         
480         for (var f in this.vrFaces) {
481             this.vrFaces[f].render (scene);
482         }
483         
484         scene.render ();
485         
486         for (var i = 0; i < this.hotspots.length; ++i) {
487             this.hotspots[i].layout ();
488         }
489         
490         this.endRender (cause, data);
491     };
492     
493     /**
494      * Render updated faces. Called as tiles are loaded from the server.
495      */
496     this.renderUpdated = function (cause, data) {
497         if (this.renderer.supportsUpdate ()) {
498             this.beginRender (cause, data);
499             
500             var scene = this.renderer.createTexturedQuadScene ();
501             
502             for (var f in this.vrFaces) {
503                 if (this.vrFaces[f].isUpdated ()) {
504                     this.vrFaces[f].render (scene);
505                 }
506             }
507             
508             scene.render ();
509             
510             for (var i = 0; i < this.hotspots.length; ++i) {
511                 this.hotspots[i].layout ();
512             }
513             
514             this.endRender (cause, data);
515         } else {
516             this.render (cause, data);
517         }
518     };

540      * Sets the mouse dragging mode.
541      *

543      */
544     this.setDragMode = function (mode) {
545         this.dragMode = mode;
546     }
547     
548     this.dragMouseDown = function (e) {
549         this.dragStart = e;
550         this.dragDistance = 0;
551     }
552     
553     this.dragMouseUp = function (e) {
554         this.dragStart = null;
555         this.smoothRotate ();
556     }
557     
558     this.dragMouseMove = function (e) {
559         if (this.dragStart != null && this.currentGesture == null) {
560             if (this.dragMode == this.DRAG_GRAB) {
561                 this.smoothRotate ();
562                 var scale = this.state.fov / this.renderer.getViewportHeight ();
563                 var dx = e.clientX - this.dragStart.clientX;
564                 var dy = e.clientY - this.dragStart.clientY;
565                 this.dragDistance += dx + dy;
566                 this.setYaw (this.getYaw () - dx * scale);
567                 this.setPitch (this.getPitch () - dy * scale);
568                 this.renderAsap ();
569                 this.dragStart = e;
570             } else {
571                 var scale = 0.1 * this.state.fov / this.renderer.getViewportHeight ();
572                 var dx = e.clientX - this.dragStart.clientX;
573                 var dy = e.clientY - this.dragStart.clientY;
574                 this.dragDistance = dx + dy;
575                 this.smoothRotate (
576                     function () {
577                         return dx * scale;
578                     },
579                     function () {
580                         return dy * scale;
581                     });
582             }
583         }
584     }

637     /**
638      * Computes the minimum field of view where the resulting image will not
639      * have to stretch the textures more than given by the
640      * {@link bigshot.VRPanoramaParameters#maxTextureMagnification} parameter.
641      *
642      * @type number
643      * @return the minimum FOV, below which it is necessary to stretch the 
644      * vr cube texture more than the given {@link bigshot.VRPanoramaParameters#maxTextureMagnification}
645      */
646     this.getMinFovFromViewportAndImage = function () {
647         var halfHeight = this.renderer.getViewportHeight () / 2;
648         
649         var minFaceHeight = this.vrFaces[0].parameters.height;
650         for (var i in this.vrFaces) {
651             minFaceHeight = Math.min (minFaceHeight, this.vrFaces[i].parameters.height);
652         }
653         
654         var edgeSizeY = this.maxTextureMagnification * minFaceHeight / 2;
655         
656         var wy = halfHeight / edgeSizeY;
657         
658         var mz = Math.atan (wy) * 180 / Math.PI;
659         
660         return mz * 2;
661     }
662     
663     this.screenToRay = function (x, y) {
664         var dray = this.screenToRayDelta (x, y);
665         var ray = this.renderer.transformToWorld ([dray.x, dray.y, dray.z]);
666         ray = Matrix.RotationY (-this.transformOffsets.yaw * Math.PI / 180.0).ensure4x4 ().x (ray);
667         ray = Matrix.RotationX (-this.transformOffsets.pitch * Math.PI / 180.0).ensure4x4 ().x (ray);
668         ray = Matrix.RotationZ (-this.transformOffsets.roll * Math.PI / 180.0).ensure4x4 ().x (ray);
669         return {
670             x : ray.e(1),
671             y : ray.e(2),
672             z : ray.e(3)
673         };
674     }
675     
676     this.screenToRayDelta = function (x, y) {
677         var halfHeight = this.renderer.getViewportHeight () / 2;
678         var halfWidth = this.renderer.getViewportWidth () / 2;
679         var x = (x - halfWidth);
680         var y = (y - halfHeight);
681         
682         var edgeSizeY = Math.tan ((this.state.fov / 2) * Math.PI / 180);
683         var edgeSizeX = edgeSizeY * this.renderer.getViewportWidth () / this.renderer.getViewportHeight ();
684         
685         var wx = x * edgeSizeX / halfWidth;
686         var wy = y * edgeSizeY / halfHeight;
687         var wz = -1.0;
688         
689         return {
690             x : wx,
691             y : wy,
692             z : wz
693         };
694     }

696     /**
697      * Smoothly rotates the panorama so that the 
698      * point given by x and y, in pixels relative to the top left corner
699      * of the panorama, ends up in the center of the viewport.
700      *
701      * @param {int} x the x-coordinate, in pixels from the left edge
702      * @param {int} y the y-coordinate, in pixels from the top edge
703      */
704     this.smoothRotateToXY = function (x, y) {
705         var ray = this.screenToRayDelta (x, y);
706         
707         var dpitch = Math.atan (ray.y) * 180 / Math.PI;
708         var dyaw = Math.atan (ray.x) * 180 / Math.PI;
709         
710         this.smoothRotateTo (this.snapYaw (this.getYaw () + dyaw), this.snapPitch (this.getPitch () + dpitch), this.getFov (), this.state.fov / 200);

713     /**
714      * Gives the step to take to slowly approach the 
715      * target value.
716      *
717      * @example
718      * current = current + this.ease (current, target, 1.0);
719      * @private
720      */
721     this.ease = function (current, target, speed, snapFrom) {
722         var easingFrom = speed * 40;
723         if (!snapFrom) {
724             snapFrom = speed / 5;
725         }
726         var ignoreFrom = speed / 1000;
727         
728         var distance = current - target;
729         if (distance > easingFrom) {
730             distance = -speed;
731         } else if (distance < -easingFrom) {
732             distance = speed;
733         } else if (Math.abs (distance) < snapFrom) {
734             distance = -distance;
735         } else if (Math.abs (distance) < ignoreFrom) {
736             distance = 0;
737         } else {
738             distance = - (speed * distance) / (easingFrom);
739         }
740         return distance;
741     }

800     /**
801      * Starts auto-rotation of the camera. If the yaw is constrained,
802      * will pan back and forth between the yaw endpoints. Call
803      * {@link #smoothRotate}() to stop the rotation.
804      */
805     this.autoRotate = function () {
806         var that = this;
807         var scale = this.state.fov / 400;
808         
809         var speed = scale;
810         var dy = speed;
811         this.smoothRotate (
812             function () {
813                 var nextPos = that.getYaw () + dy;
814                 if (that.parameters.minYaw < that.parameters.maxYaw) {
815                     if (nextPos > that.parameters.maxYaw || nextPos < that.parameters.minYaw) {
816                         dy = -dy;
817                     }
818                 } else {
819                     // The only time when minYaw > maxYaw is when the interval
820                     // contains the 0 angle.
821                     if (nextPos > that.parameters.minYaw) {
822                         // ok, we're somewhere between minYaw and 0.0
823                     } else if (nextPos > that.parameters.maxYaw) {
824                         dy = -dy;
825                     } else {
826                         // ok, we're somewhere between 0.0 and maxYaw
827                     }
828                 }
829                 return dy;
830             }, function () {
831                 return that.ease (that.getPitch (), 0.0, speed);
832             }, function () {
833                 return that.ease (that.getFov (), 45.0, 0.1);
834             });
835     }

837     /**
838      * Smoothly rotates the panorama to the given state.
839      *
840      * @param {number} yaw the target yaw
841      * @param {number} pitch the target pitch
842      * @param {number} fov the target vertical field of view
843      * @param {number} the speed to rotate with
844      */
845     this.smoothRotateTo = function (yaw, pitch, fov, speed) {
846         var that = this;
847         this.smoothRotate (
848             function () {
849                 var distance = that.circleDistance (yaw, that.getYaw ());
850                 return -that.ease (0, distance, speed);
851             }, function () {
852                 return that.ease (that.getPitch (), pitch, speed);
853             }, function () {
854                 return that.ease (that.getFov (), fov, speed);
855             }
856         );
857     }

859     /**
860      * Integer acting as a "permit". When the smoothRotate function
861      * is called, the current value is incremented and saved. If the number changes
862      * that particular call to smoothRotate stops. This way we avoid
863      * having multiple smoothRotate rotations going in parallel.
864      * @private
865      * @type int
866      */
867     this.smoothrotatePermit = 0;
868     
869     /**
870      * Smoothly rotates the camera. If all of the dp, dy and df functions are null, stops
871      * any smooth rotation.
872      *
873      * @param {function()} [dy] function giving the yaw increment for the next frame
874      * @param {function()} [dp] function giving the pitch increment for the next frame
875      * @param {function()} [df] function giving the field of view (degrees) increment for the next frame
876      */
877     this.smoothRotate = function (dy, dp, df) {
878         ++this.smoothrotatePermit;
879         var savedPermit = this.smoothrotatePermit;
880         if (!dp && !dy && !df) {
881             return;
882         }
883         
884         var that = this;
885         var stepper = function () {
886             if (that.smoothrotatePermit == savedPermit) {
887                 if (dy) {
888                     that.setYaw (that.getYaw () + dy());
889                 }
890                 
891                 if (dp) {
892                     that.setPitch (that.getPitch () + dp());
893                 }
894                 
895                 if (df) {
896                     that.setFov (that.getFov () + df());
897                 }
898                 that.render ();
899                 window.setTimeout (stepper, 1);
900             }
901         };
902         stepper ();
903     }

925     /**
926      * Translates mouse wheel events.
927      * @private
928      */
929     this.mouseWheel = function (event){
930         var delta = 0;
931         if (!event) /* For IE. */
932             event = window.event;
933         if (event.wheelDelta) { /* IE/Opera. */
934             delta = event.wheelDelta / 120;
935             /*
936              * In Opera 9, delta differs in sign as compared to IE.
937              */
938             if (window.opera)
939                 delta = -delta;
940         } else if (event.detail) { /* Mozilla case. */
941             /*
942              * In Mozilla, sign of delta is different than in IE.
943              * Also, delta is multiple of 3.
944              */
945             delta = -event.detail;
946         }
947         
948         /*
949          * If delta is nonzero, handle it.
950          * Basically, delta is now positive if wheel was scrolled up,
951          * and negative, if wheel was scrolled down.
952          */
953         if (delta) {
954             this.mouseWheelHandler (delta);
955         }
956         
957         /*
958          * Prevent default actions caused by mouse wheel.
959          * That might be ugly, but we handle scrolls somehow
960          * anyway, so don't bother here..
961          */
962         if (event.preventDefault) {
963             event.preventDefault ();
964         }
965         event.returnValue = false;
966     };
967     
968     /**
969      * Utility function to interpret mouse wheel events.
970      * @private
971      */
972     this.mouseWheelHandler = function (delta) {
973         var that = this;
974         var target = null;
975         if (delta > 0) {
976             if (this.getFov () > this.parameters.minFov) {
977                 target = this.getFov () * 0.9;
978             }
979         }
980         if (delta < 0) {
981             if (this.getFov () < this.parameters.maxFov) {
982                 target = this.getFov () / 0.9;
983             }
984         }
985         if (target != null) {
986             this.smoothRotate (
987                 function () {
988                     return 0;
989                 }, function () {
990                     return 0;
991                 }, function () {
992                     return (target - that.getFov ()) / 1.5;
993                 });        
994         }
995     };

1004     /**
1005      * Maximizes the image to cover the browser viewport.
1006      * The container div is removed from its parent node upon entering 
1007      * full screen mode. When leaving full screen mode, the container
1008      * is appended to its old parent node. To avoid rearranging the
1009      * nodes, wrap the container in an extra div.
1010      *
1011      * <p>For unknown reasons (probably security), browsers will
1012      * not let you open a window that covers the entire screen.
1013      * Even when specifying "fullscreen=yes", all you get is a window
1014      * that has a title bar and only covers the desktop (not any task
1015      * bars or the like). For now, this is the best that I can do,
1016      * but should the situation change I'll update this to be
1017      * full-screen<i>-ier</i>.
1018      *
1019      * @param {function()} [onClose] function that is called when the user 
1020      * exits full-screen mode
1021      * @public
1022      */
1023     this.fullScreen = function (onClose) {
1024         if (this.fullScreenHandler) {
1025             return;
1026         }
1027         
1028         var message = document.createElement ("div");
1029         message.style.position = "absolute";
1030         message.style.fontSize = "16pt";
1031         message.style.top = "128px";
1032         message.style.width = "100%";
1033         message.style.color = "white";
1034         message.style.padding = "16px";
1035         message.style.zIndex = "9999";
1036         message.style.textAlign = "center";
1037         message.style.opacity = "0.75";
1038         message.innerHTML = "<span style='border-radius: 16px; -moz-border-radius: 16px; padding: 16px; padding-left: 32px; padding-right: 32px; background:black'>Press Esc to exit full screen mode.</span>";
1039         
1040         var that = this;
1041         
1042         this.fullScreenHandler = new bigshot.FullScreen (this.container);
1043         this.fullScreenHandler.restoreSize = this.sizeContainer == null;
1044         
1045         this.fullScreenHandler.addOnResize (function () {
1046                 that.onresize ();
1047             });
1048         
1049         this.fullScreenHandler.addOnClose (function () {
1050                 if (message.parentNode) {
1051                     try {
1052                         div.removeChild (message);
1053                     } catch (x) {
1054                     }
1055                 }
1056                 that.fullScreenHandler = null;
1057             });
1058         
1059         if (onClose) {
1060             this.fullScreenHandler.addOnClose (function () {
1061                     onClose ();
1062                 });
1063         }
1064         
1065         this.fullScreenHandler.open ();
1066         this.fullScreenHandler.getRootElement ().appendChild (message);
1067         
1068         setTimeout (function () {
1069                 var opacity = 0.75;
1070                 var iter = function () {
1071                     opacity -= 0.02;
1072                     if (message.parentNode) {
1073                         if (opacity <= 0) {
1074                             try {
1075                                 div.removeChild (message);
1076                             } catch (x) {}
1077                         } else {
1078                             message.style.opacity = opacity;
1079                             setTimeout (iter, 20);
1080                         }
1081                     }
1082                 };
1083                 setTimeout (iter, 20);
1084             }, 3500);
1085         
1086         return function () {
1087             that.fullScreenHandler.close ();
1088         };
1089     };
