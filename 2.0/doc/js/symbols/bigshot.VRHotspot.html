Class bigshot.VRHotspot

Abstract base class for panorama hotspots. A Hotspot is simply an HTML element that is moved / hidden etc. to overlay a given position in the panorama.
Defined in: VRHotspot.js.

Field Detail
{function(clipData)} clippingStrategy
The method to use for dealing with hotspots that extend outside the viewport. Note that #CLIP_ADJUST et al are functions, not constants. To set the value, you must call the function to get a clipping strategy: 
var hotspot = ...;
// note the function call below ---------------v
hotspot.clippingStrategy = hotspot.CLIP_ADJUST ();

Method Detail
{boolean} clip(clipData)
Clips the hotspot against the viewport. Both parameters are in/out. Clipping is done by adjusting the values of the parameters.
Parameters:
clipData
Information about the hotspot.
{number} clipData.x
the x-coordinate of the top-left corner of the hotspot, in pixels.
{number} clipData.y
the y-coordinate of the top-left corner of the hotspot, in pixels.
{number} clipData.w
the width of the hotspot, in pixels.
{number} clipData.h
the height of the hotspot, in pixels.
{number} clipData.opacity Optional
the opacity of the hotspot, ranging from 0.0 (transparent) to 1.0 (opaque). If set, the opacity of the hotspot element is adjusted.
Returns:
true if the hotspot is visible, false otherwise
